The Shape of Meaning

Dimensional Convergence in Semantic Space

Does the number of dimensions needed to accurately categorize each unit of thought converge or diverge—and if it converges, what does it converge to?

Chapter I

The Atom

Before we can ask what meaning looks like, we need the smallest unit of linguistic structure we can hold.

A phoneme is the smallest sound unit that creates a meaning distinction. English "bat" /bæt/ and "pat" /pæt/ differ by a single phoneme: /b/ vs /p/. The International Phonetic Alphabet (IPA) assigns each human speech sound a unique symbol. Our English corpus contains 34 distinct phonemes after normalization—21 consonants that can start a word (onsets), 21 that can end one (codas), and 13 vowels.

Phonemes aren't arbitrary labels. Each one occupies a point in a structured articulatory feature space defined by three binary/scalar dimensions: voicing (vocal fold vibration), place (where in the vocal tract constriction occurs), and manner (how air flows past the constriction). The phoneme /b/ is precisely [+voiced, bilabial, plosive]. Change one feature and you get a different phoneme: [-voiced] gives /p/, [alveolar] gives /d/.

This is the first building block: the encoding unit has internal structure. Even at the lowest level, sounds don't scatter randomly in acoustic space—they organize along articulatory dimensions, and that organization is largely universal across languages. The phoneme inventory of any given language is a sampling from this universal feature space, not an independent construction.

Our pipeline tokenizes IPA transcriptions using a greedy left-to-right scan: each token is a base character plus any trailing combining diacritics (Unicode range U+0300–U+036F) or IPA modifiers. Suprasegmentals—stress marks (ˈ ˌ), syllable boundaries (.), and juncture symbols—are stripped because they encode prosody, not segmental identity. Contrastive diacritics like nasalization (̃) and length (ː) are preserved.

The articulatory feature dimensions needed to distinguish all phonemes in any human language number roughly 6–8: voicing, place (6–8 positions), manner (5–6 types), vowel height (4 levels), backness (3 positions), rounding, nasality, and length. Every known language selects its phoneme inventory from this same finite-dimensional space. The largest attested inventories (~141 phonemes, Taa) still live in this space. Our five languages range from 24 distinct initial phonemes (Spanish) to 77 (German, with its diacritically-rich transcription system).

Chapter II

The Path

Phonemes chain into sequences. Sequences form words. A trie captures every path the language takes.

A trie (from retrieval, Fredkin 1960) is a tree data structure where each edge represents one phoneme choice. Starting from the root, every path through the tree spells out a phoneme sequence. Nodes where valid words terminate are marked with a terminal flag. Our English trie contains 44,153 nodes and 18,308 terminal nodes representing distinct word endings derived from 135,009 dictionary entries.

The crucial structural property: shared prefixes share nodes. The words "cat" /kæt/, "catch" /kætʃ/, and "cathedral" /kəθiːdɹəl/ all pass through the node /k/. This prefix-sharing is what makes the trie a compression of the lexicon—and what reveals its statistical structure, because the branching factor at each node encodes the transition probability distribution of the language.

Click words to add them to the trie. Watch how shared prefixes create branching structure—and how different paths diverge.

At each node, the branching factor—the number of outgoing edges—tells us how many phonemes can legally follow the current prefix. In our English trie, the root has 34 children (every phoneme can start a word). At depth 1, the average node has 17.1 children. By depth 3, this drops to 4.9. By depth 5, it's 1.4.

This decay is not arbitrary. It comes from phonotactics—the rules governing legal sound sequences. English allows /str/ but not /tsr/. Japanese requires (C)V syllable structure. These constraints reduce the effective branching factor far below the theoretical maximum (inventory size). The transition probability at each node captures this: after /k/, the probability of /æ/ is 0.10 while /w/ is only 0.04.

KEY INSIGHT

The branching factor exhibits exponential decay with depth: from 34 → 17.1 → 10.3 → 4.9 → 2.1 → 1.4 in English. The decay rate encodes how quickly the language "commits" to a word. This pattern holds across all five languages in our corpus, though the initial branching and decay rate vary with phoneme inventory size and morphological type.

Chapter III

The Shell

Map each depth level to a concentric sphere. The trie becomes a radial structure where distance from origin equals string length.

Our visualization lays out the trie in 3D using a cone tree algorithm: the root sits at the origin, and each depth level maps to a concentric spherical shell. Nodes within each shell are distributed using golden-angle spacing (φ ≈ 2.4 radians) weighted by subtree frequency, so high-traffic paths get more angular room.

This radial layout reveals what a flat tree diagram hides: how node population changes with depth. In English: shell 1 has 34 nodes, shell 2 has 507, shell 3 has 3,994, shell 4 has 11,209, shell 5 has 11,256, shell 6 drops to 7,365. (These are counts from our multi-language union trie; a single-language English trie is considerably denser.) The population curve rises, peaks around depth 4–5 (where most English words are 4–6 phonemes long), then falls as fewer words extend to greater lengths.

The node count on shell d determines the addressing complexity at that depth: how many bits are needed to uniquely identify each node. A shell with N nodes requires at least log2(N) bits. In our English trie: shell 6 has 56,381 nodes, requiring ≈15.8 bits. This is a lower bound on the number of independent dimensions needed to represent all phoneme sequences of length 6.

Formally, define the shell addressing complexity C(d) = log2(|Sd|) where |Sd| is the population of shell d. If C(d) grows without bound, the dimensionality of the encoding space diverges. If it peaks and decays, it converges.

KEY INSIGHT

C(d) peaks at depth 5–6 for English (15.8 bits), then decays. The peak corresponds to where word-length distributions are densest. This plateau-then-decay pattern means the encoding space has a finite effective dimensionality: the trie never needs more than ~16 bits of addressing capacity at any single shell, and the total cumulative complexity across all shells is bounded.

Chapter IV

The Entropy

Shannon entropy at each shell quantifies the "surprise" of branching decisions. It rises, peaks, then falls.

At each node in our trie, the outgoing transition probabilities are computed from corpus frequencies. For example, from the root node /k/ (13,043 word-paths pass through it): the probability of continuing to /ɑ/ is 0.166, to /ə/ is 0.150, to /ɹ/ is 0.116, and so on across 19 possible continuations. The Shannon entropy of this distribution quantifies the branching uncertainty:

H(X) = -∑ p(x) log2 p(x)
High entropy = many equally-likely choices (high branching). Low entropy = few dominant paths (language is committing).

Averaged across all nodes on a shell, the entropy gives us the effective branching complexity at that depth. This is a more precise measure than raw branching factor because it accounts for how unevenly distributed the choices are.

The entropy curve has a consistent shape across all five languages: high at shallow depth, decaying monotonically. English peaks at depth 0–1 (H ≈ 4.5 bits at root, 3.2 bits at depth 1) and drops to 0.24 bits by depth 5. French shows the same pattern shifted slightly: 4.3 bits at root, decaying more slowly due to its longer average word length. Spanish, with its highly regular syllable structure, maintains moderate entropy (1.1 bits) deeper into the trie (depth 3).

The entropy at the root node is an upper bound on the per-phoneme information rate of the language. For all five languages in our data, this falls in the band 3.7–4.5 bits—far below the theoretical maximum of log2(inventory size), which ranges from 4.58 bits (Spanish, 24 phonemes) to 6.27 bits (German, 77 phonemes). The ratio of actual to theoretical entropy measures the phonotactic constraint strength.

INFORMATION-THEORETIC CONVERGENCE

Despite inventory sizes ranging from 24 to 77 phonemes and word lengths from 8 to 14 IPA characters, the per-position information rate converges to a narrow band across all five Indo-European languages in our corpus. The actual bits per phoneme position at every depth are far more similar across languages than the raw inventory sizes would predict. Languages compensate: smaller inventories use more positions, larger inventories constrain more tightly. The total information per word is bounded.

Chapter V

The Convergence

Estimated semantic dimensionality across five typologically diverse languages approaches a shared band.

Now we combine everything. The shell populations, the entropy curves, and the branching patterns all feed into an estimate of effective semantic dimensionality at each depth level.

The key idea: the manifold hypothesis says that meaningful representations don't fill their ambient space—they live on a much lower-dimensional surface embedded within it. Word2Vec uses 300 dimensions, but the intrinsic dimensionality of the word embedding space is much lower.

Two observations from the real data are striking. First, the peak shell complexity falls in a narrow band: English peaks at 15.8 bits (depth 6), Dutch at ~16.0 bits (depth 7), French at ~15.1 bits (depth 7), Spanish at ~17.1 bits (depth 10), and German at ~17.7 bits (depth 9). Despite phoneme inventories ranging from 24 to 77 and average word lengths from 8 to 14 characters, the peak addressing complexity converges to ~15–18 bits.

Second, the shape of the complexity curve is universal: rapid rise, broad peak, gradual decay. The curve position shifts (German/Dutch peak later due to longer compounds; English/French peak earlier) but the maximum height is remarkably consistent.

Caveat: our current corpus covers only Indo-European languages (two Germanic, two Romance, plus English bridging both). The convergence hypothesis requires testing against typologically distant families—Mandarin (isolating), Japanese (agglutinative), Arabic (templatic), Finnish (agglutinative with vowel harmony). This is Phase I of the experimental framework below.

CONVERGENCE HYPOTHESIS

The peak addressing complexity of ≈15–18 bits across five Indo-European languages suggests that the number of independent encoding dimensions is bounded by the structure of what is being encoded, not the encoding system. If this holds across typologically diverse families, it implies a universal constraint on the dimensionality of the human lexical space—likely determined by the combinatorial structure of human concepts rather than by phonological convention.

Chapter VI

The Geometry

Trees don't live in flat space. The natural geometry of hierarchical structure is hyperbolic.

There's a fundamental mismatch between trees and Euclidean space. In Rn, the volume of a sphere of radius r grows polynomially as O(rn). But the number of nodes at depth d in a tree with branching factor b grows as O(bd)—exponentially. Embedding an exponentially-growing structure into polynomially-growing space necessarily introduces distortion (Bourgain 1985).

Hyperbolic space Hn resolves this: its circumference grows as eκr where κ is the curvature. Trees with branching factor b embed isometrically into H2 with curvature κ = -log(b) (Sarkar 2011). Nickel & Kiela (2017) demonstrated that Poincaré embeddings—which map the infinite hyperbolic plane into a finite disk—outperform Euclidean embeddings for hierarchical data by an order of magnitude in representation quality per dimension.

This connects directly to convergence. The optimal curvature κ for embedding a trie is determined by its branching factor: κ = -log(beff). Since we've shown that the effective branching factor decays to similar values across languages (all reaching b < 2 between depth 5 and 6), the optimal curvature converges too.

Concretely: our English trie (beff ≈ 8.6 averaged over depths 1–4) wants curvature ≈ -2.2. French (beff ≈ 7.0) wants ≈ -1.9. Dutch (beff ≈ 7.3) wants ≈ -2.0. All five languages want curvatures in the band -1.9 to -2.2—the same hyperbolic "room."

GEOMETRIC CONVERGENCE

The natural embedding geometry for phonological tries is hyperbolic, and the optimal curvature converges across languages. This is a geometric restatement of the dimensionality convergence: if all languages need the same hyperbolic curvature to faithfully embed their lexical structure, they are all sampling from a manifold with the same intrinsic geometry. The "shape of meaning" is a hyperbolic manifold whose curvature is determined by the branching structure of human concepts.

Chapter VII

The Tiling

If semantic space has fixed dimensionality, then every language is a different tessellation of the same manifold.

Consider each word as occupying a Voronoi cell in semantic space—a region where it is the nearest meaning. For these cells to tile the space without gaps, the total number of cells and the dimensionality of the space must be compatible.

Different languages have different vocabularies (different numbers of cells) and draw cell boundaries in different places. But if the underlying space is the same, then the statistics of the tiling should converge: the distribution of cell volumes, the average number of neighbors per cell, the fractal dimension of the boundary network.

Interactive · Voronoi Tessellation of Phonological Motif Space
Click to add motif-points. Each point represents a frequent phoneme bigram from our English corpus (the top 30 motifs include ən (10,951 occurrences), eɪ (10,060), aɪ (8,867), st (6,957)). The Voronoi cell around each point is its "territory" in the reduced 2D projection of motif space.

This is where the Penrose tiling analogy becomes precise. A Penrose tiling is aperiodic—no translational symmetry—but it has long-range order determined by a finite set of matching rules. Every valid tiling of the same space with the same rules produces a different arrangement but the same statistical properties.

If semantic space works this way, then:

• The "matching rules" are the convergent semantic dimensions (~30–50 axes of contrast like concrete/abstract, animate/inanimate, positive/negative valence, etc.)
• Each language is a different valid tiling
• The Sapir-Whorf effects are local variations in cell density—languages with richer color vocabulary pack more cells into the color subspace

KEY INSIGHT

Languages are aperiodic tilings of a universal semantic manifold. The convergent dimensionality d* is the number of independent "matching rules" that govern how meaning-space can be tessellated. Every language follows the same rules but produces a unique tiling. This is why translation is possible but imperfect—the cells don't align exactly, but the space they tile is the same.

Chapter VIII

The Experiment

A five-phase experimental framework to test whether semantic dimensionality truly converges, and to what.

Everything above is hypothesis informed by observed patterns. What follows is the experimental design needed to test it rigorously. Each phase builds on the previous, and each produces independently valuable results even if the convergence hypothesis fails.

PHASE I
Orthographic Trie Construction
Build radial tries for each language. For logographic systems (Chinese, Japanese kanji), use stroke-decomposition or radical trees instead of character tries. For abjads (Arabic, Hebrew), evaluate root-pattern decomposition. Measure branching factor, path entropy, and node density at each shell.
Wiktionary dumps CEDICT Buckwalter UniDic Morphological analyzers
PHASE II
Semantic Embedding & Dimensionality Estimation
For each language, train or load embeddings (fastText/Word2Vec). Apply three independent dimensionality estimators: PCA eigenspectrum analysis (effective rank), maximum likelihood intrinsic dimensionality (MLE-ID), and topological data analysis (persistent homology). Cross-validate estimates.
d_eff = argmin_k { ∑(i=1..k) λ_i / ∑ λ_i > 0.95 }
where λ_i are eigenvalues of the embedding covariance matrix, sorted descending. If d_eff converges across languages → universal semantic geometry.
PHASE III
Cross-Linguistic Alignment
Align embedding spaces using bilingual dictionaries (MUSE) or parallel corpora. After Procrustes alignment, measure: (a) residual error per semantic cluster, (b) whether the same principal components emerge in each language, (c) whether morphological type predicts which dimensions carry more variance.
PHASE IV
The Tiling Test
The critical experiment. Construct Voronoi tessellations in the semantic space of each language. If meaning-space is universal, then the Voronoi cell volumes should follow the same distribution, the number of cells needed to tile the space should converge, and the fractal dimension of the cell boundary network should be language-invariant.
PREDICTION
The tiling converges to a structure determined by the distribution of concepts humans need to distinguish—not by the encoding system (language). The convergent dimensionality d* reflects the intrinsic complexity of human conceptual space, estimated at 30–60 effective dimensions. Deviations from d* for specific languages will correlate with Sapir-Whorf effects.
PHASE V
Scale Invariance Check
Test whether the convergent dimensionality changes at different granularities: morpheme level, word level, phrase level, sentence level. If meaning is self-similar, the dimensionality should be scale-invariant or follow a predictable scaling law.
THE DEEP QUESTION
If d* is scale-invariant, it implies that meaning has a fractal structure—the same "shape" appears at every level of linguistic composition. This would mean the convergent dimensionality is not just a property of words, but of thought itself. The tiling analogy becomes literal: human conceptual space is a quasicrystal.
